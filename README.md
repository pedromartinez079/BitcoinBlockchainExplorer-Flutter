# Bitcoin Blockchain Explorer - Flutter

An explorer for Bitcoin blockchain where transactions, blocks and addresses information can be found. This Android application was developed using Flutter/Dart. Blockchain information source: GetBlock.

## Built With
- Flutter/Dart,
- [GetBlock API, free token required](https://getblock.io/)
- [Blockchain.com API](https://www.blockchain.com/explorer/api)
- VS Code
